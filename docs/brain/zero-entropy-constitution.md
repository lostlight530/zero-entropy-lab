# 核心理念 (Core Philosophy)

# Zero-Entropy Lab | Constitution

## 0 核心愿景 (Vision)

Build a zero-entropy system: absolute zero-dependency extreme cold-start speed and fundamentalist engineering practice Code is law protocol is truth

## 1 物理架构 (Topological Rules)

- **src/kernel/**: Core logic domain Cortex Nexus Scholar
- **src/scripts/ & src/styles/**: Frontend assets
- **data/**: Persistent storage data/knowledge/ and data/memories/ are system memories tracking allowed to maintain GitHub Workflow continuity Other temporary states are strictly untracked
- **docs/**: Pure product architecture manuals no business logic

## 2 绝对法则 (The Laws)

### 2_1 零外部依赖 (Absolute Zero-Dependency)

- **Backend**: Strict prohibition of pip install Restricted to Python 3_10+ standard library
- **Frontend**: Strict prohibition of npm install Restricted to Vanilla JS ESM HTML5 CSS3 Prohibition of any bundling/build tools e_g_ Vite Webpack

### 2_2 极简工程 (Fundamentalist Engineering)

- **Cortex**: Direct sqlite3 manipulation ORM prohibited
- **Nexus**: Native http_server routing

### 2_3 公开透明 (Public Transparency)

- Keep the repository pure All personal sensitive data historical archaeological remnants and temporary caches must be cleared before Commit

## 3 协作准则 (Collaboration)

- AI partners must maintain the low-entropy state of the system Any behavior introducing complexity requires prior Review
