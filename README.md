## 📜 Project Philosophy

This laboratory operates under the **Zero-Entropy** philosophy. We aim for no third-party dependencies, minimal bloat, and lightweight native engineering.

## 🏗️ Architecture Matrix

1. **🧠 Cortex (docs/brain)**: SQLite-driven memory engine.
2. **🔌 Nexus (docs/brain/nexus.py)**: Native `http.server` API & Static Provider.
3. **👁️ Portal (index.html)**: Vanilla JS/CSS/HTML5. No compilation.
4. **⚔️ Proving Ground (tests.py)**: Native `unittest` verification.

## 🚀 Execution Commands

### Launch Laboratory (Web Portal)

```bash
python docs/brain/nexus.py serve
```

*Access at: [http://localhost:8000](http://localhost:8000)*

### Run Sensory Ingest (Scan Codebase)

```bash
python docs/brain/nexus.py ingest
```

### Verification (Tests)

```bash
python tests.py
```

## 🧬 Why Zero-Entropy?

Because restraint in design often leads to clearer architecture. In this repository, we prioritize data structure, minimal memory footprint, and cold-start efficiency.
